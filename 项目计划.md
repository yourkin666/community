# 交流社区全栈项目计划

## 📋 项目概述

**项目名称**: 交流社区平台  
**技术栈**: React + Spring Boot + MyBatis + MySQL 
**项目类型**: 全栈Web应用  

## 🎯 项目目标

构建一个简单的博客平台，支持用户登录和发布文章功能，采用简洁现代的UI设计风格。

## 🏗️ 技术架构

### 后端技术栈
- **Spring Boot 2.7+** - 主框架（不使用Spring Security）
- **MyBatis** - 数据库操作
- **MySQL 8.0** - 主数据库
- **Maven** - 依赖管理
- **原生HttpSession** - 会话管理

### 前端技术栈
- **React 18+** - 主框架
- **React Router** - 路由管理
- **Axios** - HTTP请求
- **Ant Design** - UI组件库（简洁现代风格）
- **CSS Modules** - 样式管理

### 本地开发环境
- **本地MySQL** - 数据库服务

## 📊 数据库设计

### 核心表结构

```sql
-- 用户表
CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    avatar VARCHAR(255),
    bio TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 文章表
CREATE TABLE articles (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(200) NOT NULL,
    content LONGTEXT NOT NULL,
    summary TEXT,
    author_id BIGINT NOT NULL,
    status ENUM('DRAFT', 'PUBLISHED') DEFAULT 'DRAFT',
    view_count INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (author_id) REFERENCES users(id)
);
```

## 🚀 本地开发部署

### 启动步骤
1. **数据库准备**
   ```bash
   # 安装MySQL 8.0
   # 创建数据库
   mysql -u root -p
   CREATE DATABASE community;
   # 执行初始化脚本
   source database/init.sql
   ```

2. **后端启动**
   ```bash
   cd backend
   mvn spring-boot:run
   # 或者在IDE中直接运行CommunityApplication.java
   ```

3. **前端启动**
   ```bash
   cd frontend
   npm install
   npm start
   ```

## 🎨 UI设计原则

- **简洁现代**: 采用简洁的设计风格，突出内容
- **响应式**: 支持多设备访问
- **用户体验**: 注重交互体验和操作流畅性
- **一致性**: 保持设计元素的一致性
